RGB é um projeto de estudo, cujo objetivo é testar as opções de autenticação e permissão do Django. Dependendo do usuário logado, os cards vermelho, verde e azul estarão disponíveis, assim como seus links.

Pronto Para Testar!

Usuários = user + RGB, R, G, B ou RB - senha = 123456

Os usuários terão acesso aos cards conforme sua permissão.
